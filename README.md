# Анализ лояльности пользователей Яндекс Афиши

## О проекте

В данном проекте проводится анализ поведения пользователей сервиса Яндекс Афиша. Основная задача — выявить факторы, влияющие на возврат пользователей и совершение повторных покупок.

## Цель исследования

Определить характеристики пользователей и их первого заказа, которые связаны с высокой вероятностью повторных покупок.

## Задачи

* Выгрузка данных из базы данных PostgreSQL
* Предобработка данных
* Приведение выручки к единой валюте
* Формирование профиля пользователя
* Исследовательский анализ данных
* Выявление факторов, влияющих на возврат пользователей
* Формирование рекомендаций для бизнеса

## Используемые данные

В проекте используются данные о заказах пользователей, включая:

* идентификаторы пользователей и заказов
* дату и время заказа
* тип устройства
* информацию о мероприятиях
* регион и город
* выручку и количество билетов

Дополнительно используется датасет с курсами валют для конвертации тенге в рубли.

## Основные результаты

* Большинство пользователей совершают 1–2 заказа
* Доля пользователей с высокой активностью существенно ниже
* Первый заказ влияет на вероятность возврата
* Основной канал взаимодействия — мобильные устройства
* Выявлены сегменты пользователей с более высоким уровнем retention

## Рекомендации

* Пользователи с интервалом между покупками менее 10 дней совершают больше заказов,
поэтому рекомендуется запускать триггерные коммуникации в первые 7–10 дней после покупки.
* Пользователи с 2–4 билетами в заказе демонстрируют более высокую лояльность —
стоит таргетировать их семейными и групповыми предложениями.
* Пользователи с высоким средним чеком (5+ заказов) показывают более высокий LTV —
для них можно внедрить премиальные предложения.
